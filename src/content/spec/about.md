# 关于我 / About Me

你好！我是 **千叶梦奈** ，一名始终怀抱好奇心的终身学习者，热衷于在交叉领域中探寻创新的可能。

这个博客既是我思考的容器，也是我成长的注脚。我会在这里记录学习计算机、数学、算法、物理等学科时遇到的妙趣问题与偶然洞见，也会整理项目实践与课程作业中的经验教程。除此之外，我也相信，知识不止于书本，灵感也常藏于旅途与日常之间，所以，我还会分享一些个人的见闻与经历，当作与这个世界对话的方式。

愿你在这里，找到一点共鸣，或一丝启发。
